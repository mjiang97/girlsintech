Resources for Learning Coding

MIT Open Courseware

MIT offers free course content available for you to browse through at your leisure. Choose from courses such as:

  • Introduction to Programming in Java
  • Introduction to Computer Science and Programming
  • Practical Programming in C
Khan Academy

One of the original free online coding resources, Khan Academy has come a long way. With easy-to-follow course sections with step-by-step video tutorials, Khan Academy is a great place to get started with your coding career.

Coursera

The king of online education, Coursera offers free classes from dozens of universities across the country, with a healthy smattering of coding classes for those with the desire to learn.

Code Academy

Code Academy is a well-known first stop for those looking to begin their coding education. Students can choose from several different track courses, focusing on:

  • JavaScript
  • PHP
  • Python
  • HTML + CSS
  • Ruby
  • jQuery
edX

edX offers tons of MOOCs, including courses on programming. Current upcoming programming classes include:

  • Programming Languages
  • Programming for Everybody (Python)
SoloLearn

SoloLearn provides various coding courses both on its website and on its app which allows for "on-the-go" practice and a great interactive experience.
